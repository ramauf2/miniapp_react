# Быстрая инструкция: Полноэкранный режим и отключение зума

## Что изменено

### 📱 Файлы
1. `index.html` - добавлены meta-теги и скрипт инициализации
2. `src/styles/index.css` - добавлены CSS стили для предотвращения зума
3. `src/app/App.tsx` - добавлен useEffect для инициализации Telegram WebApp
4. `src/global.d.ts` - обновлены TypeScript типы

## Результат

### ✅ Отключен зум
- ❌ Зум пинчем (двумя пальцами)
- ❌ Зум двойным тапом
- ❌ Зум при фокусе на input

### ✅ Полноэкранный режим
- ✓ Приложение занимает весь экран в Telegram
- ✓ Работает на Android и iOS
- ✓ Учитывается нижняя панель iOS Safari

### ✅ Улучшенный UX
- ✓ Отключено выделение текста
- ✓ Отключено контекстное меню
- ✓ Отключен pull-to-refresh

## Как проверить

1. Откройте приложение в Telegram на телефоне
2. Попробуйте зумить - не должно работать
3. Приложение должно быть на весь экран

## Отключение (если нужно)

### Вернуть зум
В `index.html` измените:
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0" />
```

### Отключить полноэкранный режим
В `src/app/App.tsx` закомментируйте useEffect с `tg.expand()`

## Поддержка

Все изменения совместимы с:
- ✓ Android (все версии)
- ✓ iOS (все версии)
- ✓ Telegram WebApp API
